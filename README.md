### FLUTTER 实现macos音乐播放器

> 接口感谢 [NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi) 

### 功能

- 歌曲播放  已实现
- 歌单 已实现
- 歌单 详情
- 歌曲和歌单搜索 需要优化路由和TextField问题
- FM 待实现
- 视频列表
- 视频播放 待实现

### 问题

- 个别音乐无法播放
- 没有登录有的音乐时长会是试听的片段
- TextField macos端异常bug
